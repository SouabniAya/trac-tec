Format de fichier requis

Le fichier doit respecter les critères suivants :

  • Format .txt uniquement
  • Chaque ligne doit contenir une trace d'exécution au format suivant :
CALLER:NomModule,METHOD:NomMethode,CALLEE:NomModule;

Exemple de fichier de trace :

CALLER:A,METHOD:validate,CALLEE:A; CALLER:A,METHOD:receive,CALLEE:B; CALLER:B,METHOD:receive,CALLEE:B; CALLER:B,METHOD:receive,CALLEE:B; CALLER:A,METHOD:validate,CALLEE:A; CALLER:A,METHOD:receive,CALLEE:B;
